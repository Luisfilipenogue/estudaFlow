# Changelog

## [1.0.0] — 2025-01-01

### Adicionado
- Interface gráfica com Tkinter (GUI dark-mode)
- Gerenciamento de tarefas: criar, concluir e remover
- Gerenciamento de disciplinas: criar e remover
- Vínculo de tarefas a disciplinas
- Prazo, prioridade e notas por tarefa
- Aba de resumo com estatísticas (total, concluídas, pendentes, atrasadas)
- Persistência em JSON no diretório home do usuário
- Testes automatizados com pytest (18 casos)
- Análise estática com Ruff
- Pipeline de CI com GitHub Actions
